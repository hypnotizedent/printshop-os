# Artwork & File Guidelines

## DST Files (Embroidery)
- **Lookup Protocol:** Check `DST_ARCHIVE` bucket.
- **Naming Convention:** `[domain]_[logo_type].dst` (e.g., `manecoffee_hat_logo.dst`).
- **Standard:** If a client reorders, ALWAYS check for existing DST before quoting "New Digitizing".

## Ink Colors
- **Digital (DTF/DTG):** Machine reads HEX/RGB directly. No PMS mixing.
- **Screen Print:** Requires PMS matching.
    - **Blacks:**
        - Standard Black -> Process Black
        - "Off Black" / Vintage -> Black 7C or Cool Grey 9
    - **Whites:**
        - Standard -> White
        - "Vintage White" / Cream -> PMS 7499 C (or similar cream)

## Address Placement (Mockups)
- **Left Chest:** 3.5" down from collar seam, centered between placket/center and side seam.
- **Full Back:** 4" down from collar seam.
