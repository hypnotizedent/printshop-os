## üîç **Common Problems & Solutions**

### **Problem 1: "Customizer Shows Blank"**
**Symptoms:**
- Theme customizer opens but shows nothing
- White screen or loading spinner
- No preview of website

**Solutions:**
1. **Clear browser cache** (Ctrl+F5 or Cmd+Shift+R)
2. **Try different browser** (Chrome, Firefox, Safari)
3. **Disable browser extensions** temporarily
4. **Check internet connection**

### **Problem 2: "Website Looks Broken"**
**Symptoms:**
- Website loads but looks terrible
- Missing images, broken layout
- Wrong colors or styling

**Solutions:**
1. **Check if you're editing the right theme**
2. **Revert to original Vessel theme**
3. **Start fresh with new theme upload**

### **Problem 3: "Can't Upload Theme"**
**Symptoms:**
- Upload button doesn't work
- Error message when uploading
- Theme doesn't appear in list

**Solutions:**
1. **Check file size** (should be under 50MB)
2. **Verify it's a .zip file**
3. **Try uploading through different browser**
4. **Contact Shopify support**

### **Problem 4: "Changes Don't Save"**
**Symptoms:**
- Make changes but they don't appear
- Save button doesn't work
- Changes disappear after refresh

**Solutions:**
1. **Make sure you click "Save"**
2. **Check if you're on the right theme**
3. **Try publishing the theme**
4. **Clear browser cache**
